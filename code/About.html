Gift QuestionImporter was created to help teachers create quizzes with the GIFT syntax.

GIFT is a short, intuitive and normalize syntax used in the Moodle community to create Quiz. It allows to store quiz in a format that can be versionnize, share and easily modified.

This project was made by Jonathan G.V. with the help of Christopher Fuhrman during a special project done at L'École de technologie supérieure (ETS) de Montréal. This project was modified by Esteban J. Puello F. and Professor Christopher Fuhrman at L'École de technologie supérieure (ETS) de Montréal during a Estaban's master project. You can submit issues, requests and comments on the github repository : GIFT-google-quiz-plugin.
